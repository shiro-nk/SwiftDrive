Detailed Description

Handler / File + Text Format Object Array:
Handles the reading and writing of HandleType compatible objects to and from A file. This is similar to Data by providing a way to add, remove, get, set, and find elements in an array. However, rather than strings, Handlers works with HandleTypes and is centered around file I/O.

Ryan Kerr
